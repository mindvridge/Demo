### 02_타로앱_QuickStart_Guide.md ⭐⭐⭐⭐
**읽기 시간**: 1시간
**대상**: 신규 개발자

**주요 내용**:
- **개발 환경 설정**
  - Node.js, npm, Git
  - React Native 환경
  - IDE 설정 (VS Code)

- **저장소 클론 및 설치**
  ```bash
  git clone [repo-url]
  npm install
  ```

- **로컬 서버 실행**
  - 백엔드 서버 실행
  - 프론트엔드 개발 서버
  - 데이터베이스 연결

- **개발 워크플로우**
  - Git Flow 전략
  - 브랜치 네이밍
  - 커밋 메시지 규칙
  - 코드 리뷰 프로세스

- **주요 명령어**
  ```bash
  npm run dev        # 개발 서버
  npm run test       # 테스트
  npm run build      # 빌드
  npm run deploy     # 배포
  ```

### 03_타로앱_MVP_실행체크리스트.md ⭐⭐⭐⭐
**읽기 시간**: 1시간
**대상**: 전체 개발팀

**주요 내용**:
- **200+ 실행 항목 체크리스트**

**카테고리별 체크리스트**:

1. **프로젝트 설정** (10개)
   - Git 저장소 생성
   - CI/CD 파이프라인
   - 개발 환경 구성

2. **백엔드 개발** (50개)
   - 사용자 인증 API
   - 타로 카드 API
   - 리딩 API
   - 결제 API

3. **프론트엔드 개발** (60개)
   - 홈 화면
   - 리딩 플로우
   - 프로필 화면
   - 설정 화면

4. **AI/ML** (20개)
   - GPT-4 통합
   - 해석 생성 엔진
   - 추천 시스템

5. **디자인** (30개)
   - 디자인 시스템
   - 컴포넌트 라이브러리
   - 애니메이션

6. **테스트** (20개)
   - 단위 테스트
   - 통합 테스트
   - E2E 테스트

7. **배포** (10개)
   - AWS 인프라
   - 앱스토어 제출
   - 모니터링
