## 2. 타겟 고객 분석

### 2.1 주요 타겟 페르소나

#### Primary Target: "영적 탐구자" (25-34세)
- **특성**: 자기계발 관심, MBTI 애호가, 정신건강 중시
- **니즈**: 자기이해, 관계 조언, 커리어 가이드
- **행동**: 일일 루틴화, 소셜 공유 활발
- **지불의사**: 월 1-3만원

#### Secondary Target: "캐주얼 사용자" (18-24세)
- **특성**: 호기심 중심, 트렌드 민감, 친구와 공유
- **니즈**: 재미, 연애운, 일상 조언
- **행동**: 무료 콘텐츠 선호, 바이럴 참여
- **지불의사**: 건당 소액 결제

### 2.2 사용자 여정 (User Journey)

```
발견 → 첫 경험 → 습관화 → 유료전환 → 충성고객
```

1. **발견**: SNS 바이럴, 친구 추천, 앱스토어
2. **첫 경험**: 무료 일일 타로, 가입 보너스
3. **습관화**: 푸시 알림, 일일 리워드
4. **유료전환**: 프리미엄 기능 체험, 할인 이벤트
5. **충성고객**: 커뮤니티 참여, 추천 보상

## 3. 제품 전략

### 3.1 핵심 차별화 요소

#### 1. AI 하이퍼 개인화
- **행동 학습**: 사용자 패턴 분석으로 정확도 향상
- **컨텍스트 인식**: 시간, 계절, 개인 일정 반영
- **감정 AI**: 텍스트/음성 감정 분석

#### 2. 한국형 통합 점술
- **동서양 융합**: 타로 + 사주 + 별자리 통합 분석
- **K-Culture**: 한국 정서와 문화 반영
- **로컬라이제이션**: 절기, 명절 특별 운세

#### 3. 소셜 & 커뮤니티
- **타로 매칭**: 궁합/호환성 분석
- **그룹 리딩**: 친구/연인과 함께하는 타로
- **스토리 공유**: 익명 경험 공유 커뮤니티

### 3.2 MVP 기능 설계

#### Phase 1: Core Features (출시 시점)
- ✅ AI 일일 타로 리딩
- ✅ 78장 카드 데이터베이스
- ✅ 3가지 기본 스프레드
- ✅ 리딩 히스토리 저장
- ✅ 기본 개인화 (생년월일, 성별)

#### Phase 2: Growth Features (3개월)
- ➕ 심화 스프레드 (10종)
- ➕ 음성 질문 입력
- ➕ 감정 분석 기반 카드 추천
- ➕ 주간/월간 운세 리포트
- ➕ 소셜 공유 기능

#### Phase 3: Premium Features (6개월)
- 💎 실시간 전문가 상담
- 💎 AR 타로 체험
- 💎 개인 타로 덱 커스터마이징
- 💎 관계 호환성 분석
- 💎 AI 코칭 & 목표 추적

### 3.3 기술 스택

#### Frontend
- **Native**: Swift (iOS) / Kotlin (Android)
- **Cross-platform 대안**: Flutter or React Native
- **Web**: Next.js + TypeScript

#### Backend
- **서버**: Node.js + Express / Python FastAPI
- **데이터베이스**: PostgreSQL + Redis
- **AI/ML**: TensorFlow / PyTorch
- **클라우드**: AWS / Google Cloud

#### AI 모델
- **자연어 처리**: GPT-4 API / Claude API
- **이미지 생성**: Stable Diffusion / DALL-E
- **감정 분석**: 자체 학습 모델
- **추천 시스템**: 협업 필터링 + 콘텐츠 기반
