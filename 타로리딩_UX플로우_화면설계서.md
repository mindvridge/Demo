## 3. 인터랙션 상세

### 3.1 제스처 & 터치

| 제스처 | 액션 | 피드백 |
|--------|------|--------|
| 탭 | 카드 선택 | 햅틱 + 글로우 |
| 롱프레스 | 카드 상세 보기 | 확대 + 설명 |
| 스와이프 | 다음/이전 카드 | 슬라이드 전환 |
| 핀치 | 스프레드 전체 보기 | 줌 인/아웃 |
| 더블탭 | 카드 뒤집기 | 3D 플립 |

### 3.2 애니메이션 타이밍

```javascript
const AnimationTimeline = {
  // 카드 셔플
  shuffle: {
    start: 0,
    cardMovement: 300,  // 각 카드 0.3초
    totalDuration: 3000,
    easing: "easeInOutCubic"
  },
  
  // 카드 선택
  selection: {
    hover: 100,
    select: 200,
    glow: 500,
    lift: 300
  },
  
  // 카드 배치
  placement: {
    fly: 800,          // 날아가기
    settle: 200,       // 자리잡기
    stagger: 400       // 카드 간 딜레이
  },
  
  // 카드 공개
  reveal: {
    flip: 600,

    highlightKey: 300
  }
};
```

## 9. 플로우 최적화 제안

### 9.1 단축 경로

```
빠른 리딩 모드:
홈 → 질문 입력 → 즉시 3카드 → 해석

전문가 모드:
홈 → 스프레드 선택 → 상세 설정 → 리딩
```
