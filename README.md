# TarotMind 프로젝트 문서 가이드

## 📚 문서 구조 개요

이 저장소는 TarotMind(AI 타로 앱) 프로젝트의 모든 기획, 설계, 개발 문서를 포함합니다.
문서들은 **개발 단계별로 폴더에 정리**되어 있으며, 각 파일명 앞에 **순서 번호**가 붙어 있어 순차적으로 읽을 수 있습니다.
