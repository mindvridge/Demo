## 📋 상세 Sprint 계획 (Stage 1: MVP)

### Sprint 1-2: 기초 설정 (Week 5-8)

#### Sprint 1 (Week 5-6): 인프라 & 인증
**개발 태스크**:
- [ ] AWS/GCP 인프라 구축
  - EKS/GKE 클러스터 설정
  - PostgreSQL RDS 설정
  - Redis 클러스터 설정
- [ ] 백엔드 기본 구조
  - NestJS 프로젝트 초기화
  - GraphQL 스키마 설계
  - JWT 인증 구현
- [ ] 사용자 관리 API
  - 회원가입/로그인
  - 소셜 로그인 (Google, Apple, Kakao)
  - 프로필 CRUD

**테스트**:
- Unit tests: 인증 로직
- Integration tests: API 엔드포인트
- 커버리지 목표: 80%

**Definition of Done**:
- ✅ 사용자 가입 및 로그인 가능
- ✅ JWT 토큰 발급 및 검증 작동
- ✅ API 문서 자동 생성 (Swagger)

#### Sprint 2 (Week 7-8): 타로 데이터 & 엔진
**개발 태스크**:
- [ ] 타로 카드 데이터베이스
  - 78장 카드 데이터 입력
  - 카드 이미지 S3 업로드
  - 카드 의미 JSON 구조화
- [ ] 타로 엔진 코어
  - 카드 셔플 알고리즘

  - 카드 선택 로직
- [ ] 기본 해석 시스템
  - GPT-4 API 통합
  - 프롬프트 엔지니어링
  - 해석 생성 파이프라인

**테스트**:
- 셔플 랜덤성 검증
- AI 응답 속도 테스트 (< 3초)
